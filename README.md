

**Student Management System**

* This is a simple and user-friendly Student Management System developed using HTML, CSS, and JavaScript.
* The application helps in managing student records with basic and practical features suitable for internship-level projects.

* The main goal of this project is to focus on clean UI, proper validation, and real-world frontend logic rather than complex frameworks.

---

Features

* Add new student records
* Edit existing student details
* Delete student records
* Search and filter students by name or roll number
* Inline form validation with clear error messages
* Roll number validation (numeric only)
* Dark and light mode toggle
* Data stored using browser LocalStorage
* Responsive and clean user interface

---

Technology Used

* HTML5 for structure
* CSS3 for styling and layout
* JavaScript (Vanilla) for logic and DOM manipulation
* LocalStorage for client-side data persistence

How to Run the Project

1. Download or clone the project folder
2. Open the index.html file in any modern web browser
3. Start adding, editing, and managing student records

No installation or external libraries are required.
